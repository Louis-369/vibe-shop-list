# ✈️ 世界口袋 To Buy (World Pocket To Buy)

這是一個專為旅行者設計的行動優先購物清單 App。不需要安裝任何 App，只需一個 HTML 檔案即可在瀏覽器中運行，並支援 PWA 安裝至手機主畫面。

## ✨ 最新功能 (v3.5)

- **🔢 數量計算**: 支援輸入商品數量，自動計算總價（單價 × 數量），購物預算更精準。
- **📍 購買地點記憶**: 新增購買地點欄位，系統會自動記憶您常用的店家，下次輸入時可快速選擇。
- **🖼️ 圖片網址貼上**: 除了拍照/上傳，現在支援直接貼上圖片網址，整理清單更方便。
- **💰 價差分析**: 完成購買輸入實際金額後，系統會自動計算並顯示「省了多少」或「貴了多少」。
- **📉 排序優化**: 新增依照「購買地點」排序，方便在同一家店一次買齊。

## 🌟 核心特色

- **📱 行動優先**: 針對單手操作優化，支援滑動手勢 (左滑刪除、右滑購買)、震動回饋。
- **💱 即時匯率**: 整合 API 抓取匯率，自動換算 JPY, KRW, USD, EUR 等幣別至 TWD。
- **🎨 24 節氣主題**: 獨家動態主題系統，隨季節與節氣自動變換 App 色調。
- **📋 智慧清單**:
  - **批次貼上**: 自動辨識文字中的商品名稱與金額。
  - **自訂排序**: 支援拖曳調整順序。
  - **購物助手**: 智慧標籤建議 (如：免稅、尺寸、顏色)。
- **💾 資料安全**: 資料儲存在瀏覽器本地 (LocalStorage)，支援 JSON 檔案或代碼備份/還原。

## 🛠️ 技術棧

單頁應用程式 (SPA)，無需後端：

- **HTML5 / PWA**
- **Tailwind CSS** (CDN)
- **JavaScript (Vanilla)**
- **SortableJS** (拖曳排序)
- **FontAwesome** (圖示)

## 🚀 如何使用

1.  用瀏覽器 (Chrome, Safari) 開啟 https://louis-369.github.io/vibe-shop-list/。
2.  **手機用戶**: 推薦使用「加入主畫面」功能，獲得原生 App 般的體驗。
